# yany8060.github.io

## 安装Hexo
> 默认安装了node.js

### Installation
```
$ npm install hexo-cli -g
```

### Quick Start
```
$ hexo init blog
$ cd blog
```

### Start the server
```
$ hexo server
```

### Create a new post
```
$ hexo new "Hello Hexo"
```

### Generate static files
```
$ hexo generate
```

### Clean static files
```
$ hexo clean
```

### Deploy static files
```
$ hexo deploy
```
